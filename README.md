# AI-Automation-for-Employment-Certificate-Processing

## Docker & Docker Compose
To run compose:
```
docker compose up

-d                              (detached)
--build                         (with build)
```
To stop compose:
```
docker compose down

-v                              (with networks and volumes)
```

To clean:
```
docker volume prune             (clean volumes)
docker system prune -af         (clean all)
```
